Правила игры

  • Администратором задаются параметры игрового поля в формате «морской бой» - квадрата размерностью NхN.
  • Корабли могут быть только однопалубными.
  • За каждым кораблем должен быть закреплен приз, без этого корабль нельзя создать и разместить на поле.
  • Нельзя ставить корабли в одну и ту же ячейку
  • Каждый пользователь должен иметь свой уникальный ID.
  • Добавление пользователя на игровое поле производится по его уникальному идентификатору.
  • Удаление игрока с поля производится по его уникальному идентификатору.
  • Администратор должен предусмотреть начисление выстрелов для пользователя на конкретном игровом поле.
  • Количество выстрелов у одного и того же пользователя на различных игровых полях может отличаться.
  • Важно, администраторы могут только начислять выстрелы пользователям. Стрелять по игровому полю администраторы не могут.
